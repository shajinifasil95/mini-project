# Restaurant and Food Inspections Analysis

## Project Overview
This Power BI project analyzes restaurant inspection data to identify food safety violations, risk levels, and inspection trends.

## Tools Used
- Excel
- Power BI
- Power Query
- DAX

## Dashboard Features
- KPI Cards
- Risk Analysis
- Violation Trends
- Map Visualization
- Interactive Filters
